# shop-ui

UI web virtus-shop: menampilkan produk, form beli (POST order), dan daftar order.
Dilayani di path `/shop` lewat shop-gateway; browser memanggil `/api/products`
dan `/api/orders` same-origin via gateway yang sama (tanpa CORS).

Microservice ke-4 — ditambahkan untuk membuktikan pola: repo baru + folder baru
di gitops, ApplicationSet mendeteksi otomatis, pipeline PaC identik.
